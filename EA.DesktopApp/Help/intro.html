Введение

Программный продукт Ffind предназначен для распознования лиц и сохранения персональной карточки человека в базе данных. Программа позволяет распозновать лица в реальном времени в видеопотоке.

При работе с данной программой необходимо настроить сервер и базу данных. База данных поставляется с программным продуктом.

Программа имеет удобный и простой интерфейс, а также подробную справочную информацию и всплывающие подсказки.

Программный продукт Ffind распространяется и используется по лицензии.

Продукт полностью совместим с операционной системой Microsoft Windows начиная с версии 7.
