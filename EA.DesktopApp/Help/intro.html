Introduction

The Ffind software product is designed to recognise faces and store a person's personal card in a database. The software allows you to recognise faces in real time in a video stream.

When working with this program, it is necessary to configure the server and database. The database is supplied with the program product.

The program has a user-friendly and simple interface, as well as detailed help information and tooltips.

The Ffind software product is distributed and used under license.

The product is fully compatible with the Microsoft Windows operating system from version 7 onwards.
