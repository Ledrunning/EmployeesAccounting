Запуск и работа с приложением

Для запуска приложения необходимо осуществить двойное нажание левой кнопкой мыши на иконке приложения.

После запуска приложения откроется основное окно, в котором происходит распознование лица в видеопотоке захватываемого с видеокамеры.

В левом нижнем углу показывается реальное время компьютера, на котором установлено приложение

Для запуска видеозазвата необходимо нажать кнопку "Старт"

Старт - позволяет запустить режим видеозахвата и распознования лиц.

Сделать фото - открытие нового окна для создания фотографии ее сохранения в локальную директори или заполнение карты персоны и добавление в базу данных.

Для заполнения карты сотрудника и получения фотографии необходимо нажать кнопку "Сделать фото"

Старт - позволяет запустить режим видеозахвата и распознования лиц.

Сделать фото - открытие нового окна для создания фотографии ее сохранения в локальную директори или заполнение карты персоны и добавление в базу данных.

Для заполнения карты сотрудника и получения фотографии необходимо нажать кнопку "Сделать фото"

После чего откроется новое окно в котором можно в режиме видеозахвата создать фотографию

Фото - позволяет сделать фотографию.

Добавить - добавление в базу данных.

Удалить - удаление сотрудника из базы данных.

Сохранить - сохранение цветной фотографии в локальную директорию.

Поля для заполнения

Имя - поле для ввода имени сотрудника, обязательно для заполнения.

Фамилия - поле для ввода фамилии сотрудника, обязательно для заполнения.

Департамент - поле для ввода департамента сотрудника, обязательно для заполнения.

В случае если оператор не нажал кнопку "Фото", соответственно не сделал фотографию и пытается сохранить нажав кнопку "Сохранить", система сообщит оповещением в виде диалогового окна.

В случае если хоть одно поле для ввода карты сотрудника будет не заполнена система оповестить оператора диалоговым окном о необходимости заполнения формы

Вызов справки осуществляется нажатием на кнопку со знаком вопроса либо клавишей F1
