Starting and working with the application

To launch the application, double-click with the left mouse button on the application icon.

After launching the application, the main window will open, in which the face is recognized in the video stream captured from the video camera.

The bottom left corner shows the real time of the computer where the application is installed

To start the video capture you should press the "Start" button

Start - button allows you to start the video capture and face detection mode.

Photo - button opening a new window to create a photo and save it to a local directory or fill in a employee personal data and add it to the database.

Use the "Photo" button to complete the employee personal data and obtain a photo

After opening a new window you will see a video capture frame for photos and fields for entering personal data as well as three control buttons.

Photo - allows you to take a picture.

Add - adding to the database.

Delete - deleting an employee from the database.

Edit - edit personal data of the selected person.

Fields to be filled in/P>

Name - field for entering the employee's name, must be filled in.

Last name - field for entering the employee's surname, must be filled in.

Department - field for entering the employee's department, must be filled in.

In case the operator has not pressed the "Photo" button, respectively has not taken a photo and tries to save it by pressing the "Save" button, the system will inform with a notification in the form of a dialog box.

If at least one field for entering an employee card is not filled in, the system will notify the operator with a dialog box about the need to fill in the form

You can call the help by pressing the button with a question mark or the F1 key
